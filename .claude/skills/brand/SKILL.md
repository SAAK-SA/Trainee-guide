# Brand Agent Skill

A Claude agent that manages brand identity, voice, and consistency across your organization.

## Core Capabilities

Handles **brand voice definition**, **visual identity standards**, **messaging frameworks**, and **asset management**. Designed to help with content tone guidance, style guide development, and brand compliance reviews.

## Key Features

**Quick operational commands:**
- Inject brand context into prompts via `inject-brand-context.cjs`
- Validate assets using `validate-asset.cjs`
- Extract and compare colors against your palette with `extract-colors.cjs`

**Sync workflow** keeps your brand guidelines aligned across systems — syncing from `docs/brand-guidelines.md` to design tokens in both JSON and CSS formats.

## When to Activate

Use this skill for tasks like establishing tone of voice, developing style guides, organizing marketing assets, managing color palettes and typography, and conducting brand consistency audits.

## Available References

Accesses guidance documents covering voice frameworks, visual identity standards, messaging strategies, consistency checklists, asset organization, logo usage rules, and approval workflows — providing comprehensive brand governance support.
