# Banner Design Skill

A comprehensive multi-format banner creation system for social media, advertising, web, and print applications.

## Core Capabilities

Generates **multiple art direction options** per request using AI-powered visuals. Handles banner design exclusively — not video editing, full website design, or print production.

## Key Workflow Steps

**1. Requirements Gathering:** Collects purpose, platform/dimensions, content, brand guidelines, style preference, and quantity via user questions.

**2. Research & Art Direction:** Activates design intelligence, researches Pinterest for references, and selects 2-3 complementary styles.

**3. Design & Generation:** Creates HTML/CSS banners with exact platform dimensions, applies brand context, and generates visual elements using AI models (Standard for fast iterations; Pro for detailed hero visuals).

**4. Export to Images:** Screenshots HTML banners at precise dimensions using Chrome DevTools, auto-compresses if needed.

**5. Present & Iterate:** Displays all options side-by-side with rationale and file paths for user feedback.

## Supported Platforms & Styles

**Platforms:** Facebook, Twitter/X, LinkedIn, YouTube, Instagram (story/post), Google Display, website hero, print.

**Styles:** Minimalist, gradient, bold typography, photo-based, illustrated, geometric, retro, glassmorphism, 3D, neon, duotone, editorial, collage.

## Key Design Standards

Critical content stays within the central 70-80% safe zone. Banners enforce max 2 typefaces, single CTA, and 4.5:1 contrast ratio. Text should remain under 20% of ad space to comply with platform policies.
