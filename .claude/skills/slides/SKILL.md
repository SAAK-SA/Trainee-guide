# Slides Skill: Strategic HTML Presentation Design

Creates data-driven presentations using HTML with Chart.js visualization, responsive layouts, and strategic copywriting approaches.

## Primary Use Cases

Works best for marketing decks, pitch presentations, and content requiring visual data representation. Designed for professionals building "strategic HTML presentations with Chart.js, design tokens, responsive layouts, copywriting formulas, and contextual slide strategies."

## How It Works

The system follows a three-step routing process:
1. Identifies the subcommand from your input (first word)
2. Retrieves the corresponding reference documentation
3. Executes using remaining arguments

## Available Resources

Four reference materials support slide creation:
- **Layout Patterns** — Structural templates for organizing visual hierarchy
- **HTML Template** — Base markup for presentation structure
- **Copywriting Formulas** — Message frameworks for persuasive content
- **Slide Strategies** — Context-specific approaches to slide design

## Getting Started

Use the `create` subcommand with your topic and desired slide count: `[topic] [slide-count]`

This generates professionally structured presentations optimized for both visual impact and persuasive messaging.
