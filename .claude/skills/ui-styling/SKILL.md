# UI Styling Skill

A comprehensive skill for building modern user interfaces using three complementary technologies.

## Core Technology Stack

- **shadcn/ui**: Pre-built accessible components based on Radix UI primitives with a copy-paste distribution model
- **Tailwind CSS**: A utility-first styling framework offering build-time processing with zero runtime overhead
- **Canvas-based design**: "Visual communication over text" for sophisticated visual compositions

## Primary Use Cases

Deploy this skill when developing React-based applications (Next.js, Vite, Remix, Astro) that require:
- Accessible component implementation
- Responsive, mobile-first layouts
- Dark mode functionality
- Consistent design systems
- Rapid prototyping with visual feedback

## Quick Implementation

Setup via CLI (`npx shadcn@latest init`), component installation, and practical code examples demonstrating form validation, responsive grids, and dark mode patterns.

## Knowledge Organization

References specialized guides covering component catalogs, theming systems, accessibility patterns, responsive design, and customization strategies. Includes Python automation scripts for component installation and configuration generation.

The philosophy emphasizes "expert craftsmanship" treating UI development as a disciplined practice where composition guides attention and every detail matters.
